# Legacy Mechanics

This directory contains the previous five production Challenge Family implementations retained as internal mechanic infrastructure:

- Scene Investigation
- Flash Words
- Spot the Difference
- Object Recall
- Pattern Recall

They are not the player-facing Story Mode identity. The production runtime may still load them through `src/gameplay/families/manifest.json` while future Witness Moments decide how and when to use their mechanics.

Do not delete or redesign a family here without a migration plan, replacement mechanic contract, save/progression review, and regression validation.
