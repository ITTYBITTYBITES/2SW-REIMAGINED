# Rank Progression Evolution

## Current foundation

Production progression currently provides XP, levels, mastery, accuracy, streaks, achievements, unlocks, programs, profile persistence, and recommendations.

These values remain authoritative. This document defines their future player-facing identity.

## Identity-based progression

### Rank 1 — Observer

Theme: **I Notice**

Perception meaning:

- remain with a field;
- notice a meaningful detail;
- complete the first Witness Moments.

### Rank 2 — Witness

Theme: **I See**

Perception meaning:

- compare perspective;
- recognize change;
- make evidence legible.

### Rank 3 — Chronicler

Theme: **I Preserve**

Perception meaning:

- retain moments;
- build an Archive;
- connect present discoveries to history.

### Rank 4 — Weaver

Theme: **I Connect**

Perception meaning:

- connect evidence;
- recognize relationships;
- understand longer Story Chapters.

Future ranks remain reserved.

## Migration mapping

| Current system | Future presentation |
|---|---|
| XP | Witness Rank progress |
| Numeric level | Rank/Chapter eligibility |
| Family mastery | Perception mastery |
| Achievements | Milestones/Discoveries |
| Streaks | Consistency / Daily Witness continuity |
| Unlocked families | New perception/story locations |
| Programs | Story Chapter paths |
| Recent history | Witness Archive |
| Personal stats | Your Iris records |

## Rules

- Do not reset XP, levels, mastery, streaks, achievements, or unlocks.
- Do not create a second rank save system.
- Rank names are presentation metadata until migration is approved.
- A rank should unlock a way of seeing, not only a menu category.
- Incorrect results can still contribute to understanding/recovery according to production policies.
