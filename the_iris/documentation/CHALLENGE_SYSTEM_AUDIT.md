# Challenge System Audit — Transition Note

The existing five production families remain available as legacy mechanic providers. This blueprint does not redesign them.

- Scene Investigation
- Flash Words
- Spot the Difference
- Object Recall
- Pattern Recall

They are no longer intended to be the player-facing Story Mode product categories. Future Witness Moments should select perception mechanics invisibly and frame them through observation, memory, discovery, investigation, understanding, and emotional storytelling.

The detailed family/runtime audit remains in this file’s history. The transition strategy is defined in `LEGACY_CHALLENGE_TRANSITION_PLAN.md`; future mechanic containers are defined in `FUTURE_WITNESS_MECHANICS_FRAMEWORK.md`.
