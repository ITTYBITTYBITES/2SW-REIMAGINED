# Player Emotion Guide

A Witness Moment should create presence, curiosity, understanding, and reflection. It should not rely primarily on shock, violence, murder, horror, jump scares, or humiliation.

```text
Curiosity
→ Wonder / Beauty / Empathy
→ Suspense / Uncertainty
→ Discovery
→ Relief / Understanding
→ Reflection
→ Hope / Bittersweet meaning / Quiet triumph
```

The player should leave thinking: “I was there. I saw something that mattered.”
