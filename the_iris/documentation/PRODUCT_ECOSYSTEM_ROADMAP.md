# Product Ecosystem Roadmap

## Current foundation

- Iris navigation foundation
- Production challenge runtime
- Five challenge families
- Saves/profile/settings/progression
- Accessibility/audio/analytics
- Mobile Simulator and device adaptation
- Story/Daily/Weekly/Your Iris placeholders

## Future roadmap shape

### Update 1 — Scene Investigation Story Moment

Build the first complete Iris → Observation → Recall → Evidence → Reward → Iris journey using existing Scene Investigation mechanics.

### Update 2 — Witness Rank / Rank Chapters

Frame existing XP/mastery/unlocks as Observer, Investigator, Archivist chapters without replacing production progress calculations.

### Update 3 — Witness Archive

Unify production history, evidence, achievements, completed moments, and replay/tutorial entries.

### Update 4 — Daily Witness

Add an offline deterministic daily selection layer using existing recommendations, saves, and family runtime.

### Update 5 — Weekly Investigation

Add a local weekly investigation contract with progress, resume, and Archive reward.

### Update 6 — Spot the Difference Moment

Convert existing comparison mechanics into a cinematic evidence/change-detection Witness Moment.

### Update 7 — Object Recall Moment

Convert set memory into a collection/absence Witness Moment.

### Update 8 — Pattern Recall Moment

Convert sequence/path mechanics into a connection/Archivist moment.

### Update 9 — Flash Words Signal Moment

Convert typography mechanics into a fleeting-signal moment with reading comfort.

### Update 10 — Ecosystem depth

Expand Programs, seasonal content, discovery relationships, and long-term mastery after the core Story Mode language is proven.

## Roadmap rules

- Each update is additive and documented.
- No update replaces production contracts without migration.
- Each update receives a dedicated feature audit, implementation plan, validation report, and rollback strategy.
- New content should be usable by Story Mode, Library, Daily/Weekly, or Archive without a new root navigation hierarchy.
