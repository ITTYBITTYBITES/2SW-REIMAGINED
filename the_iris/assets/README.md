# Two Second Witness Asset Pipeline

Phase 0 prepares placeholder locations for future production assets. These folders are intentionally empty except for `.gitkeep` files; no final artwork is introduced in this phase.

- `scenes/` — future scene-stage artwork, generated-scene backplates, and ordinary-world visual sets.
- `evidence/` — future reveal annotations, evidence focus art, and comparison support assets.
- `home/` — future Witness Home preview imagery and invitation art.
- `record/` — future Witness Record archive/personal-history presentation assets.
- `branding/` — future shell-level identity assets that are distinct from existing app icons and brand marks.

Production assets should remain offline-capable, token-aware for contrast where applicable, and documented in the relevant flagship update report before release.
